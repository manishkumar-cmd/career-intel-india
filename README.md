# 🎯 Career Intelligence App – Python Edition

A **100% Python** career analysis tool built with Streamlit for the Indian job market.
Perfect as a **Data Science portfolio project**.

---

## ✨ What's Inside

| File | Purpose |
|------|---------|
| `app.py` | Main Streamlit application |
| `data/database.py` | All job market data (6 roles, salaries, skills) |
| `algorithms/scoring.py` | Job fit score algorithm |
| `algorithms/salary.py` | Salary estimation engine |
| `algorithms/recommender.py` | Personalized recommendations |
| `utils/charts.py` | All Plotly visualizations |
| `utils/styling.py` | Custom CSS for professional UI |


## ⚠️ Disclaimer

All predictions are estimates based on general Indian job market trends (2024-2025).
Actual salaries and outcomes vary by company, industry, and individual performance.
